# 🚀 PloxB - Минималистичный веб-браузер на Python

Современный, легковесный и настраиваемый веб-браузер, созданный с нуля с использованием Python и мощной библиотеки PySide6.

![Скриншот браузера PloxB]([ВСТАВЬТЕ ССЫЛКУ НА СКРИНШОТ ЗДЕСЬ])

## ✨ О проекте

**PloxB** — это учебный проект, демонстрирующий возможности создания полнофункционального десктопного приложения на Python. Браузер имеет современный, бесшовный интерфейс без стандартных рамок окна, поддержку вкладок и смену тем. В основе лежит движок QtWebEngine, который является оберткой над Chromium, что обеспечивает отличную совместимость с современными веб-стандартами.

### Основные возможности:
*   **🖥️ Современный интерфейс:** Кастомная бесшовная рамка окна и минималистичный дизайн.
*   **📑 Навигация по вкладкам:** Поддержка нескольких вкладок с возможностью их создания и закрытия.
*   **🎨 Смена тем:** Встроенный переключатель между темной и светлой темами с мгновенным применением.
*   **⚙️ Сохранение настроек:** Возможность задать домашнюю страницу и выбрать тему, которые сохраняются между сессиями.
*   **🧭 Стандартная навигация:** Кнопки "Назад", "Вперед", "Обновить" и "Домой".
*   **🔍 Умная адресная строка:** Работает и для прямых URL, и для поисковых запросов (через DuckDuckGo).

## 🛠️ Используемые технологии

*   **Python 3:** Язык программирования.
*   **PySide6:** Официальная обертка Python для фреймворка Qt, используемая для создания всего графического интерфейса.
*   **QtWebEngine:** Компонент Qt для встраивания веб-контента, основанный на Chromium.
*   **QSS (Qt Style Sheets):** "CSS для Qt" для полной кастомизации внешнего вида.

## ⚙️ Установка и запуск

Чтобы запустить PloxB на своем компьютере, следуйте этим шагам.

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/your-username/PloxB.git
    cd PloxB
    ```
    *(Не забудьте заменить `your-username` на ваш никнейм на GitHub)*

2.  **Установите зависимости:**
    Убедитесь, что у вас установлен Python и `pip`. Выполните команду:
    ```bash
    pip install -r requirements.txt
    ```
    *(В нашем случае `requirements.txt` содержит только `PySide6`)*

3.  **Запустите приложение:**
    ```bash
    python main.py
    ```
    Или используйте созданный ярлык, если вы настроили его для запуска без командной строки.
